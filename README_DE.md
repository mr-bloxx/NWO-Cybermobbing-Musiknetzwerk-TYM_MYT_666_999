# NWO-Cybermobbing-Musiknetzwerk-TYM_MYT_666_999

## Projektübersicht

Dieses Repository enthält umfassende Forschungsdokumentation und Analyse der deutschen Musik Künstler TYM und MYT, mit Fokus auf ihre vollständigen Diskografien, kollaborativen Werke, Streaming-Analysen und ihre Positionierung innerhalb der deutschen Musiklandschaft.

## Forschungsumfang

### Primäre Subjekte
- **TYM** (Tim Rumpelt): Deutscher Rapper, Produzent und Songwriter
- **MYT**: Kollaborativer Künstler, der ausschließlich mit TYM arbeitet
- **Zeitraum**: 2019-2026 (vollständige Karrieredokumentation)
- **Geografischer Fokus**: Deutschsprachige Musikmärkte

### Forschungsmethodik
Dieses Projekt employs systematische Datensammlungsmethoden einschließlich:
- Digital-Plattform-Analysen (Spotify, YouTube)
- Web-basierte Forschung und Querverweise
- Netzwerkanalyse von Industriebeziehungen
- Streaming-Performance-Metriken
- Kollaborative Musteranalyse
